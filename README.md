# Fastfetch (RorkOS Fork)

> **⚠️ WARNING:** This is **NOT** the official Fastfetch repository. This is a temporary fork maintained by the RorkOS team for logo integration and testing.

---

### Overview
This repository was created specifically to prepare and test the **RorkOS** branding for the Fastfetch tool. Our goal is to ensure a high-quality ASCII art representation for RorkOS users.

### Status
* ✅ **RorkOS Logo:** Added (Standard version).
* ⏳ **Upstreaming:** We are currently waiting for the official [fastfetch-cli/fastfetch](https://github.com/fastfetch-cli/fastfetch) maintainer to review and merge the RorkOS logo into the main branch.

Once the logo is officially merged into the upstream repository, this fork will be deprecated.
